## 🌟 About The Portfolio

This is my personal developer portfolio built from scratch to showcase my projects, skills, and technical journey. It is designed with a **Cyberpunk / Hacker aesthetic**, featuring deep dark mode contrasts, neon glass-morphism, and a completely responsive layout.

<p align="center">
  <img src="https://media.giphy.com/media/qgQUggCGvnP19XcgRX/giphy.gif" width="400" alt="Coding GIF" style="border-radius: 10px;" />
</p>

### ✨ Key Features
- **🌓 Advanced Themes:** Beautiful sleek Light Mode and a stunning Neon Dark Mode.
- **⚡ Performance First:** Built with Vite and React for lightning-fast page loads.
- **🎬 Cinematic Animations:** Custom sequential scroll reveals and smooth entry transitions.
- **🎯 3D Interactive UI:** Complex layout utilizing interactive tilt cards, custom cursors, and ambient glass-morphism.
- **🧑‍💻 Integrated Lottie:** Includes a rich, moving 2D graphical hacker workspace animation.

## 🚀 Quick Setup

Want to run this portfolio locally? Follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/piyushrana31/portfolio-me.git
   ```

2. **Navigate into the directory:**
   ```bash
   cd portfolio-me
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser** and visit `http://localhost:5173/`.
